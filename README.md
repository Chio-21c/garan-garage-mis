# Garan Garage MIS

Garan Garage MIS (Management Information System) is a web-based application designed to streamline garage operations, manage customer records, track services, and handle billing efficiently.  

---

##features
- Customer registration and management
- Vehicle service tracking
- Billing and payment records
- User authentication and role-based access
- Dashboard for quick insights

---

##Technologies Used
- **PHP** – Backend logic and server-side scripting
- **MySQL** – Database management and storage
- **CSS** – Styling and responsive design
- **Node.js** – Real-time features and API integration
